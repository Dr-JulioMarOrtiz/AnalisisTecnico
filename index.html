Módulo 07
Punto de Equilibrio Multi-Producto
Calcula el PE ponderado cuando tu empresa maneja más de un producto o línea de negocio.
ℹ️
¿Cómo funciona? Cuando vendes varios productos, el PE no puede calcularse con una sola fórmula. Se usa la Mezcla de Ventas: la proporción en que se vende cada producto. El resultado es el PE ponderado total de la empresa.

Módulo 08
Exportar Informe Completo
Descarga todos los cálculos en un documento HTML o PDF con tu información personalizada.

🌐 Descargar como HTML

RECOMENDADO
✅ Genera un archivo .html que se abre en cualquier navegador. Incluye todos los datos, tablas y cálculos con el mismo diseño visual de la herramienta. No requiere software adicional.
  • Todos los módulos en un solo archivo
  • Apto para imprimir desde el navegador
  • Compatible con Chrome, Edge, Firefox, Safari
  • Se guarda en tu carpeta de Descargas

📑 Descargar como PDF

ℹ️ El PDF se genera abriendo el informe HTML en el navegador y usando la función de impresión (Ctrl+P → "Guardar como PDF"). Es el método más confiable para mantener el formato.
  1. Haz clic en "Descargar Informe HTML"
  2. Abre el archivo descargado en Chrome o Edge
  3. Presiona Ctrl + P (o Cmd + P en Mac)
  4. Selecciona "Guardar como PDF"
  5. Elige el tamaño Carta y haz clic en Guardar
✅El HTML fue descargado. Ábrelo y usa Ctrl+P para guardar como PDF.
